Form and Textarea
Initialize editor and set the editor to WYSIWYG mode.
open SimpleForm.html  
setEditMode wysiwyg  
waitForEditMode wysiwyg  
Original content area should be removed from screen and WYSIWYG editor should be displayed.
verifyStyleDisplay xqEditor none
verifyStyleDisplay css=div.wysiwyg_editor block
verifyStyleDisplay css=div.source_editor none
verifyRawWysiwygContent [p]Hello World![/p]  
Modifying content.
setRawWysiwygContent [p]Hello World, modified[/p]  
Modified content should be applied before submit performed.
click submitButton  
assertAlert <p>Hello World, modified</p>
