For example, P which is contained by LI and is the only child is the unnecessary paragraph. But if given container element is a block-only-container(BLOCKQUOTE, BODY), this method does nothing.
Parameters:
{Element} element
Container element
Returns:
{boolean} True if unwrap performed.
