Edit Mode Toggling
Initialize editor and set the editor to WYSIWYG mode.
open SimplePage.html  
setEditMode wysiwyg  
waitForEditMode wysiwyg  
And toggle edit mode...
toggleSourceAndWysiwygMode    
Then source editor should be turned on.
verifyStyleDisplay css=div.wysiwyg_editor none
verifyStyleDisplay css=div.source_editor block
verifyStyleDisplay xqEditor none
And toggle again...
toggleSourceAndWysiwygMode    
Then WYSiWYG editor should be turned on.
verifyStyleDisplay css=div.wysiwyg_editor block
verifyStyleDisplay css=div.source_editor none
verifyStyleDisplay xqEditor none
